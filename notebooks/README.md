# Notebooks Overview

This folder contains all analytical notebooks developed for the NorthStar Urban Mobility and Logistics Analytics Project.

Each notebook focuses on a specific stage of the analytics and system development process, including SQL analytics, R visualisation, Python-based processing, and MongoDB implementation.

## Notebook Structure

### 01_sql_in_r.ipynb
This notebook contains SQL queries executed within the R environment using the sqldf package. The analysis focuses on operational efficiency, delivery performance, customer complaints, and profitability analysis.

### 02_r_analytics.ipynb
This notebook contains statistical analysis and data visualisation using R libraries such as dplyr and ggplot2. Multiple charts and operational insights were generated from the logistics datasets.

### 03_python_processing.ipynb
This notebook contains Python-based preprocessing, NumPy analysis, and data transformation tasks used to prepare operational datasets for advanced analytics.

### 04_mongodb_development.ipynb
This notebook demonstrates MongoDB CRUD operations, aggregation pipelines, indexing strategies, and query benchmarking for operational data management.

## Purpose

The notebooks collectively demonstrate the integration of SQL, R, Python, and MongoDB technologies to support data-driven operational analysis and decision-making within urban mobility and logistics systems.
