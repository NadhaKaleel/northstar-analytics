# NorthStar Urban Mobility and Logistics Analytics Project

This repository contains the complete analytical development work carried out for the NorthStar Urban Mobility and Logistics Analytics Project.

The project focuses on analysing operational efficiency, customer complaints, delivery performance, profitability, and infrastructure reliability using SQL, R, Python, and MongoDB technologies.



# Project Objectives

- Analyse operational inefficiencies across logistics and delivery services
- Identify patterns in customer complaints and delivery failures
- Evaluate profitability across different operational zones
- Perform statistical analysis and visualisation using R
- Conduct data preprocessing and transformation using Python
- Implement MongoDB CRUD operations and aggregation pipelines
- Apply query optimisation and benchmarking techniques



# Technologies Used

# Analytical Components

### SQL in R Analytics
SQL queries executed within R using the sqldf package for operational analysis and reporting.

### R Analytics and Visualisation
Statistical analysis and visualisation using dplyr, ggplot2, and plotly libraries.

### Python Data Processing
Data preprocessing, transformation, and numerical analysis using Python and NumPy.

### MongoDB Development
CRUD operations, aggregation pipelines, indexing, and query optimisation using MongoDB.



# Project Outcome

The project successfully demonstrated how data-driven analytics can improve operational efficiency, decision-making, customer service performance, and profitability within large-scale urban mobility and logistics environments.
