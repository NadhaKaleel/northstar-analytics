# Data Folder Overview

This folder contains all datasets used for the NorthStar Urban Mobility and Logistics Analytics Project.

The datasets support SQL analytics, R visualisation, Python-based processing, and MongoDB development activities carried out throughout the project.

## Contents

- delivery_operations.csv
- customer_complaints.csv
- fleet_metrics.csv
- route_performance.csv

## Purpose

The datasets were used to analyse operational efficiency, customer satisfaction, delivery failures, battery performance, and profitability across different service zones.
